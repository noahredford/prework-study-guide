HTML

  • HTML is a language that gives a webpage structure.
  • The head element contains information about the website, which is not viewable by the user.
  • The body element represents the visible content that is shown on the webpage. The user can use and interact with the body element.
  • HTML stands for Hyper Text Markup Language.
  • The tag <br> is the tag for a line break.
  • The <em> tag is used to emphasize text.
  • The tag for adding a background color is <body style="background-color:yellow;">
  • The tag for adding creating a hyperlink is <a href="http://www.website.com">website description</a>
  • The tag for creating a new, blank tab is <a href="urltarget="_blank">
  • The tag following tags are table tags <table>, <tr>, and <td>
  • The tag for creating a checkbox is <input type="checkbox">
  • The tag that allows the user to enter text into a text box is <input type ="text">
  • The tag, <select>, creates a dropdown list.
  • The tag, <strong>, defines impportant text.
  • Inline elements are normally displayed without starting a new line (inline elements only occupy the space bound by the tags defining element, they only stay on that one line.)
  • The tag <textarea>, makes a text area.
  • To instert an image, you use the tag: <img src="image.gif" alt="My Image">.
  • To insert a background image, you use the following: <body style="background-image:url(background.gif)">.
  • The <iframe> tag displays a webpage within a webpage.
  • Block elements are NOT usually displayed without starting a new line.
  • The <title> defines the title of the HTML document.
  • Alt - defines the alternate text for an image if it is not available.
  • Contenteditable - defines whether or not a specific elemet should be editable or not.
  • Onblur and Onfocus are event attributes.
  • Graphics defined by SVG are in the XML format.
  • The <canvas> tag is used to draw graphics.
  • Required - this attribute specifies whether an input field must be filled out.
  • The range output defines a slider control.
  • The <meter> tag displays a scalar measurement within a range.
  • The <nav> tag defnines navigation links.
  • The <aside> tag sets content aside from the page content.
  • The <header> tag specifies a header for a document or section.

CSS

  • CSS is a language that gives a webpage style and color for the user.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside the element.
